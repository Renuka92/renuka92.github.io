Silhoutte

This project involved using Illustrator to outline a silhouette of a dynamic figure, which in this case, is an athlete. The key to making the silhouette look so sharp was the intricate use of the pen tool to get every curve and every detail. I also added colors to distinguish different parts of the figure, which added to how vivid the entire silhouette is.
